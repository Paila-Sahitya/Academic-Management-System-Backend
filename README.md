# Academic Course & Student Performance Management System

## Project Overview
This project is a secure backend-based academic management system developed during internship training. It provides role-based authentication and academic record management functionalities.

## Features
- User Authentication (JWT Based)
- Role-Based Access Control (Admin, Faculty, Student)
- Course Management
- Course Enrollment
- Marks Management
- Attendance Tracking
- Performance Analytics

## Tech Stack
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- bcrypt
- Postman (Testing)

## Security
- Password hashing using bcrypt
- Token-based authentication using JWT
- Role-based route protection
